# Adonis AdminLTE

## Commands

```
- cp .env.example .env
- npm install
- adonis key:generate
- adonis migration:run --seed
- npm run buildDev (vai rodar um negocio e dps vai ficar atualizando sempre que editar um arquivo, mas se quiser pode cancelar pq é só pra gerar os css/js da interface)
- npm start
```
